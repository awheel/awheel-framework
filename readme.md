# Light framework

Official [Web Site](http://gitlab.hupu.com/light/light-framework)

## Light PHP Framework

小型框架, 从 HupuTv 项目衍生出来.

框架核心强大简洁, 不参与任何业务, 所需任何功能均可自定义扩展.

目前应用: [HupuTv](http://hupu.tv), [NBA比赛中心](https://nba.hupu.com/games)
