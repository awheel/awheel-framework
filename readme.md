# Awheel framework

一个小型框架, 从[虎扑Tv](http://hupu.tv)项目衍生出来.

框架核心强大简洁, 不参与任何业务, 所需任何功能均可自定义扩展.

## 使用中的项目
- [虎扑Tv](http://hupu.tv)
- [NBA比赛中心](https://nba.hupu.com/games)
- 虎扑足球 App
- [虎扑微信服务](https://wechat.hupu.com)

## Core

- 路由
- 命令行
- 控制器
- 中间件
- 组件
- 配置
- 输入&输出
